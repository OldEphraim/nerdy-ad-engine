## 1. CTA dimension structurally capped for awareness ads

The spec mandates that awareness-goal ads use "Learn More" as their CTA button. The evaluator
rubric scores CTAs on specificity, urgency, and low-friction action — "Learn More" is inherently
generic by that rubric, so awareness ads consistently score 5-6 on the `call_to_action` dimension
regardless of how strong the rest of the ad is. This is a real tension between spec compliance
(correct CTA for funnel stage) and CTA score optimization (the evaluator wants "Start your free
practice test"). We chose spec compliance — matching CTA to funnel stage is the right production
behavior, even though it costs ~1 point on the CTA dimension for half the library.

## V2 Image Layer Limitations

Image generation adds approximately 4–8 seconds of latency per ad (2 variants at ~2s each).
At concurrency=5, the full v2 pipeline takes roughly 4–6x longer than v1's text-only run.
This is the primary cost of adding the visual layer — not API spend (which only ~4x's), but
wall-clock time.

fal.ai CDN URLs expire in approximately 1 hour. The pipeline downloads images immediately
to `data/images/` to avoid broken references in the dashboard or library. `data/images/` is
gitignored because image files are too large for version control; images are regenerated by
re-running `pnpm generate`. If the pipeline is interrupted mid-run, any images already
downloaded will persist, but the incomplete ad entries won't appear in the library.

There is no iterative improvement loop for images. Unlike text, where the evaluator can say
"your CTA is weak" and `strategies.ts` provides a targeted fix, there's no well-defined
per-dimension intervention strategy for image generation. A/B variant selection — generating
2 images with different seeds and picking the higher-scoring one — is the image quality
mechanism. This is a deliberate tradeoff: an iterative loop would roughly triple image
generation cost with minimal expected quality gain (see Decision 17 in `DECISION_LOG.md`).

Visual evaluation is inherently more subjective than text evaluation. "Brand consistency" is
the hardest dimension to calibrate because we don't have official Varsity Tutors brand
guidelines, style guides, or reference creatives for the image layer. The evaluator uses
inferred brand values (empowering, knowledgeable, approachable, warm color palette) which
may not match actual brand standards. "Visual engagement" and "text-image coherence" are
somewhat more objective — composition quality and message alignment are easier to assess
without brand-specific references — but scores on these dimensions should still be treated
as directional rather than absolute.
